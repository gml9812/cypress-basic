## Getting Started

### Prerequisites
1. npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Repo를 clone한다
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. NPM pakage들을 설치한다
   ```sh
   npm install
   ```
3. CORS 해결을 위해 http-server 등을 이용해 프로젝트를 실행한다
   ```sh
   npm install http-server
   http-server
   ```
